# ClientStack Support

This repository contains the official support information and policies for **ClientStack**.

## Support

If you need help, have questions, or want to report a bug, contact us:

- **Email:** support@easycrmsoft.com

When reporting an issue, please include:
- App version (e.g., 26.1.0) and device model / iOS version
- Steps to reproduce
- Expected vs. actual result
- Screenshots or screen recordings (if possible)

## Common Topics

- **Business card scanning:** If a field is not detected correctly, please share a screenshot of the scan result screen (hide sensitive info if needed).
- **Import / Export:** If an import fails or exported data looks incorrect, include the file type (CSV/JSON/etc. if applicable), approximate record count, and any error message shown.
- **Data & storage:** ClientStack is designed to keep your data on-device. Exporting is user-initiated.
